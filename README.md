reopen
======

The Repo+Open Project Framework Manifest

build
-------

```sh
mkdir reopen && cd reopen
repo init -u https://github.com/rerepo/reopen
repo sync
source build/setupenv.sh
lunch
make
```
Then you can find fw in `out` dir
